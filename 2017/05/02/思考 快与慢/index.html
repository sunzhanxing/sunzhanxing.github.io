思考，快与慢-读书笔记

基本信息

一、大脑中的两个系统

  • 系统1（快思考）
  • 系统2（满思考）

系统1和系统2的协作

系统1先出面，而不能饶过系统1直接面对系统2。系统2分非常懒拖，它通常不会主动干预系统1的工作，除非系统1发出求助。这些就导致我们做出的很多决策其实都是一览系统1的直觉反应，而不是经过系统2的理性分析于计算，我们设置都一无所知。

二、大脑思考机制引发的心里效应

  • 1曝光效应
  • 2启动效应
  • 3常态理论
  • 4因果关系错觉
  • 5确认偏误
  • 6光环效应
  • 7框架效应

三、大脑思考机制带来的直觉缺陷

3.1启发法

  • 3.1.1 经典性启发
  • 3.1.2 合取谬误
  • 3.1.3 忽视回归均值现象
  • 3.1.4 锚定效应
  • 3.1.5 效用层叠
  • 3.1.6 结果偏见
  • 3.1.7 有消息错觉
  • 3.1.8 过度信赖直觉
  • 3.1.9 规划谬误
  • 3.1.10 乐观偏差
  • 3.1.11沉没成本论

3.2前景理论

  • 3.2.1确定效应：再确定的收益和“赌一把”之间，人们往往会选择确定的收益。
  • 3.2.2反射效应：在确定的损失和”赌一把“之间，人们往往会选择”赌一把“。
  • 3.2.3损失厌恶：人们对损失对比获得要更敏感得多。
  • 3.2.4可能性效应于确定性效应。
  • 3.2.5联合评估于偏好逆转。

3.3禀赋效应

四、应对直觉缺陷的一些方法

  • 1假设任一重要谈判中的数字都对自己有锚定效应，从而有意识地调动系统2取分析、辨别，得出真正合适的数字。
  • 2有意识地排除无效信息（干扰性暗示信息），采用回归均值的眼光进行预测，以防止盲目冒险。
  • 3善用框架效应，营造更容易获得肯定的情景。
  • 4努力养成参考、采纳外部意见的决策习惯，而不只是依赖个人的内部意见。
  • 5事前验货：预先设想惨败结果，并分析可能的原因，以此来部分抑制乐观偏见。

五、两个自我

  • 1.峰钟定律
  • 2.体验效用！=决策效用
  • 3.经验自我！=记忆自我
