高效休息法-读书笔记

Author: Pidan Date: June 2, 2018  14:50:00

基本信息

  • 作者：久贺谷亮
  • 类别：管理
  • 推荐指数：🌟🌟🌟🌟🌟

前言

虽然什么都没做，但总觉得很累……

即使周末好好睡一觉，周一早上起来还是会感到身心疲惫……

注意力没有以前集中，很快就会分心……

如果你不出现以上症状，说明你累的不是身体，而是大脑。

收获

本书透过“消除大脑疲劳的七个休息法”，以呼吸和冥想来释放压力，达到正念的目的，让大脑获得真正的休息。

一、什么是高效休息法

二、消除大脑疲劳的七个休息法

1.感觉脑袋昏昏沉沉时：正念呼吸法

效果：

  • 减轻压力
  • 抑制杂念
  • 提高注意力和记忆力
  • 控制情绪
  • 改善免疫力

步骤

1.1 采取基本坐姿

1.1.1 坐在座椅上（稍微停止背部，背部离开椅背）

1.1.2 腹部放松，双手放在大腿上，双腿不交叉。

1.1.3 闭上眼睛（如果采用睁着眼睛的方式，则双眼不聚焦地望向前方2米左右的位置）

1.2 有意识地关注身体的感觉

1.2.1 感受与周围环境的接触（脚底与地板，臀部与座椅，手与大腿等）

1.2.2 身体背重力吸引的感觉。

1.3 关注呼吸

1.3.1 关注与呼吸相关的感觉。

1.3.2 不必深呼吸或可以控制呼吸（建议使用鼻腔呼吸，等待呼吸自然到来）

1.4 如果浮现杂念

1.4.1 意识到已浮现杂念的事实，将注意力转回“专注呼吸”这件事上来。

1.4.2 产生杂念是很正常的，不必苛责自己。

2.心事重重时：动态冥想

效果

  • 提高集中中力和注意力
  • 达到新流的状态

步骤

2.1 步行冥想

2.1.1 自己决定步行的速度，但建议刚开始时走慢一点。

2.1.2 有意识地留意手脚肌肉、关节的动作，以及与地面接触的感觉。

2.1.3 给自己的动作贴上标签，例如“右和左，上和下”等

2.2 以站姿进行动态冥想

2.2.1 站着并将双脚打开至与肩同宽，向两侧伸出双臂缓慢抬高。

2.2.2 专注于手腕肌肉等变化、血液下流的感觉，并感受重力。

2.2.3 双臂抬高后，再缓慢放下（重复的动作）

3.压力导致身体状态不佳时：压力呼吸化法

效果：

  • 消除压力
  • 消除压力造成的紧张
  • 改善其他身体不适

步骤：

3.1 意识到压力带来的影响

3.1.1 采取正念呼吸法的基本姿势

3.1.2 将造成压力的原因总结成“一句话”，在心里默念这句话。

3.2 将意识集在呼吸上

3.2.1给呼吸贴上“1“ “2”等标签

3.2.2 感受身体的紧张逐渐得到舒缓

3.3 将意识扩散至全身

3.3.1将注意力扩散至全身

3.3.2 呼吸时，设想对压力有反应的身体部位在“吸气”随着呼吸，让该部分逐渐放松。

3.3.3 继续将注意力扩散至周围的空间

4.想跳脱思考怪圈时：“猴子思维”消除法

效果：

  • 抑制思考的闭环
  • 提高集中力
  • 避免自我嫌恶
  • 改善睡眠质量
  • 深度睡眠

步骤：

4.1 舍弃

4.1.1 给思考贴上标签，意识到“已经想了很多遍”的事实

4.2 考虑例外

4.2.1 反复出现同意想法，是否因为设置了同样的前提

4.2.2 尝试考虑一下这种想法之外的情况

4.3 从贤者的角度考虑

4.3.1 如果是自己所尊敬的人或历史上的为人，他们会如何考虑

4.3.2 是否将“杂念本身”与“心怀杂念的自己”等同视之一

4.4 不要判断好坏

4.4.1 是否以“当下”意外的基准来评价事物

4.4.2 时刻保持“不做评价”

4.5 探究原因

4.5.1 这种想法反复出现的原因是什么

4.5.2 从自己的“深层愿望”开始重新思考

5.被愤怒冲动冲昏头脑时：RAIN法

效果：

  • 平息怒气
  • 控制欲望
  • 抑制冲动
  • 减肥
  • 戒烟

步骤：

5.1Recognize（认知）

5.1.1 认识到自己的内心升起的愤怒

5.1.2 不将”愤怒“与“愤怒的自己”等同视

5.2Accept（接受）

5.2.1 接受自己愤怒的事实

5.2.2 不对这一事实加以价值评价，允许其存在

5.3Investigate（调查）

5.3.1 观察愤怒时身体有何变化

5.3.2 心率变化如何

5.3.3 身体哪里感到紧张

5.4Non-Identification（保持距离）

5.4.1 不将自己的情绪当作个人的问题

5.4.2 剥离愤怒，试着把愤怒当作“他人之事”

6.身体不适有痛感时：扫描全身法

效果：

  • 压力性疼痛
  • 皮肤病、热潮红
  • 调节自律神经

步骤：

6.1平躺并关注自己的呼吸

6.1.1 也可以坐在座椅上

6.1.2 还要关注伴随呼吸胸腹上下起伏的感觉

6.2将注意力集中在左脚尖

6.2.1 脚底接触鞋袜的感觉如何

6.2.2 脚趾于相邻脚趾之间触碰的感觉如何

6.3扫描全身

6.3.1 从左脚脚尖开始“扫描”

6.3.2 吸气时，由鼻腔吸入空气，经由全身，通往左脚脚尖。

6.3.3 呼气时，聚集于左脚脚尖的空气，经由全身，由鼻腔呼出

6.4全身各个部分都可以这么做

6.4.1 由左脚脚尖开始至左腿全体的扫描结束后，将同样的流程用于右腿，左、右手臂，腹部及头部等部位

6.4.2观察有痛感，等身体部位，并”扫描“这一部位

7.看他人不顺眼时：温柔的慈悲心

效果：

  • 抑制对他人的负面情绪
  • 培养正面情绪

步骤：

7.1保证正念的意识状态

7.1.1 将日常的正念呼吸法持续10分钟

7.1.2 从消极情绪回归到专注于“当下”

7.2想起那个“让你不爽”的人

7.2.1 想起造成压力的那个人的样子

7.2.2 专注于身心的变化

7.3在心中对他（她）默念一下句子

7.3.1 希望你远离各种危险，平平安安

7.3.2 希望你幸福喜乐，安心自在

7.3.3 希望你身体健康
